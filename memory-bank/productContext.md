# Contexto del Producto

## Problema / Oportunidad
El Mundial 2026 genera una alta demanda de interacción social. Los usuarios buscan competir con amigos y compañeros mediante quinielas, pero exigen una experiencia fluida, moderna y con datos al instante sin tener que refrescar la pantalla manualmente.

## Objetivos del Producto
- Ofrecer una plataforma web móvil-first intuitiva y adictiva para pronosticar partidos.
- Proveer actualizaciones de marcadores y tablas de posiciones en tiempo real (WebSockets).
- Mostrar de forma clara estadísticas de juego y el estado del torneo (incluyendo equipos eliminados).

## Estrategia de UX/UI
- **Enfoque:** Mobile-First (diseño óptimo para smartphones).
- **Visibilidad:** Equipos eliminados con opacidad reducida (escala de grises).
- **Interactividad:** Botones rápidos `+` y `-` para predicciones de goles.
